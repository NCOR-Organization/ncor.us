# National Center for Ontological Research (NCOR)

**Official website:** https://ncor.us  
**Canonical domain:** https://ncor.us  

The National Center for Ontological Research (NCOR) is an organization dedicated to advancing applied ontology, semantic interoperability, knowledge representation, and formal ontology engineering.

This repository hosts the official public website for NCOR.

---

## About NCOR

NCOR supports research, education, and professional development in:

- Applied Ontology
- Formal Ontology (including Basic Formal Ontology – BFO)
- Knowledge Graph Engineering
- Semantic Web Technologies
- Ontology Governance and Standards
- Machine–Machine Interoperability
- AI and Ontology Integration

NCOR works across academic, governmental, and industry contexts to promote rigorous ontological tradecraft and interoperable semantic infrastructure.
